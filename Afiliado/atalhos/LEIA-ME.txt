MEGA DESCONTOS - ATALHOS

Use estes arquivos para operar o projeto sem mexer nos arquivos internos.

1. instalar_automacao_completa.bat
   Execute uma unica vez neste computador.
   Configura Mercado Livre e Amazon, inicia os agentes e faz com que eles
   abram automaticamente junto com o Windows.
   Depois disso, o uso diario e feito somente pelo botao "Rodar bot agora"
   no painel do site.

2. iniciar_site.bat
   Abre o site local em http://127.0.0.1:8000

3. automatizar_mercado_livre_online.bat
   Gera links do Mercado Livre e cadastra no site online.

4. automatizar_shopee_online.bat
   Le um CSV baixado manualmente na Shopee, gera links afiliados e cadastra no site online.

5. abrir_gerador_mercado_livre.bat
   Abre somente o gerador do Mercado Livre em navegador controlavel.

6. automatizar_amazon_online.bat
   Busca ofertas na Amazon, monta links com sua tag e cadastra no site online.

7. abrir_gerador_shopee.bat
   Abre somente o gerador da Shopee em navegador controlavel.

8. gerar_links_mercado_livre.bat
   Usa o gerador ja aberto para criar CSV de links Mercado Livre.

9. gerar_links_shopee.bat
   Le um CSV baixado manualmente na Shopee e usa o gerador para criar CSV de links afiliados.

10. rodar_bot.bat
   Faz o fluxo local completo em um clique:
   - inicia o site se estiver fechado;
   - roda o bot no banco atual;
   - abre o site e a fila de revisao;
   - abre produtos monitorados, status, ofertas geradas e links candidatos.

Arquivos de entrada importantes:
- bot\links_promocoes_potenciais.txt
- bot\links_shopee_promocoes_potenciais.txt
- CSV BatchProductLinks baixado na Shopee

Arquivos de saida importantes:
- bot\links_afiliados_gerados.csv
- bot\links_shopee_afiliados_gerados.csv
- bot\links_amazon_afiliados_gerados.csv
AUTOMACAO EM UM CLIQUE - MERCADO LIVRE

Execute uma vez:
  instalar_agente_mercado_livre.bat

Depois disso, o agente inicia junto com o Windows. O botao "Rodar bot agora"
coleta as ofertas, o agente abre o Link Builder, gera os links afiliados e
publica os produtos no site. Na primeira abertura do navegador, faca login
no Mercado Livre usando o perfil aberto pelo agente.

AUTOMACAO EM UM CLIQUE - AMAZON

Depois de configurar o agente principal, execute uma vez:
  instalar_agente_amazon.bat

O agente Amazon usa um navegador e perfil separados, busca ofertas atuais,
monta os links com a tag megadesco0304-20 e publica junto com as outras lojas
quando voce clicar em "Rodar bot agora".

ARCH LINUX / KDE PLASMA

No Arch Linux, instale uma vez as dependencias do sistema:
  sudo pacman -S python python-pip chromium

Depois, dentro da pasta Afiliado, execute:
  chmod +x atalhos/*_linux.sh
  ./atalhos/instalar_automacao_linux.sh

O instalador Linux:
- cria um ambiente Python local em .venv;
- instala as dependencias Python;
- configura os agentes;
- cria entradas de inicializacao em ~/.config/autostart para o KDE Plasma;
- inicia Mercado Livre, Amazon e Magalu.

Uso diario no Linux:
  ./atalhos/iniciar_agentes_linux.sh

Ou simplesmente deixe o KDE abrir os agentes automaticamente quando voce
entrar na sessao. O botao "Rodar automacao completa" no painel online vai
enviar trabalho para esses agentes, igual no Windows.

Na primeira abertura do navegador, faca login no Mercado Livre usando o perfil
aberto pelo agente. Amazon e Magalu usam perfis separados.
