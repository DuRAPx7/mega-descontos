MEGA DESCONTOS - ATALHOS

Esta pasta agora fica separada por fluxo atual:

WINDOWS - USO DIARIO

1. INICIAR_TODOS_AGENTES_WINDOWS.bat
   Use este arquivo no Windows.
   Ele abre 3 janelas separadas:
   - Mercado Livre
   - Amazon
   - Magalu

   Deixe as janelas abertas ou minimizadas.
   Depois clique em "Rodar automacao completa" no painel online.


WINDOWS - AGENTES INDIVIDUAIS

2. iniciar_agente_mercado_livre.bat
   Inicia somente o agente Mercado Livre.

3. iniciar_agente_amazon.bat
   Inicia somente o agente Amazon.

4. iniciar_agente_magalu.bat
   Inicia somente o agente Magalu.


WINDOWS - CONFIGURACAO

5. instalar_automacao_completa.bat
   Execute quando for configurar tudo pela primeira vez no Windows.
   Ele configura Mercado Livre, Amazon e Magalu.

6. instalar_agente_mercado_livre.bat
   Configura somente Mercado Livre.

7. instalar_agente_amazon.bat
   Configura somente Amazon.

8. instalar_agente_magalu.bat
   Configura somente Magalu.


SITE LOCAL

9. iniciar_site.bat
   Abre o site local em http://127.0.0.1:8000.
   Normalmente voce usa o painel online:
   https://mega-descontos.onrender.com/admin.html


LINUX / ARCH / KDE PLASMA

10. instalar_automacao_linux.sh
    Instala/configura os agentes no Linux.

11. iniciar_agentes_linux.sh
    Inicia todos os agentes no Linux.

12. iniciar_agente_mercado_livre_linux.sh
13. iniciar_agente_amazon_linux.sh
14. iniciar_agente_magalu_linux.sh
    Iniciam os agentes individualmente no Linux.


IMPORTANTE

- O painel online nao abre programas sozinho no Windows.
- Primeiro abra os agentes com INICIAR_TODOS_AGENTES_WINDOWS.bat.
- Depois rode a automacao pelo painel.
- Se uma janela fechar ou mostrar erro, tire print dela.
